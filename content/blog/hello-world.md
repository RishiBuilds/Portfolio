Welcome to my personal AI & Software Developer Portfolio!

This site is built using **Next.js 15**, **Base UI**, and **Tailwind CSS**. To make content editing extremely frictionless, I implemented a development-time file system watcher using `chokidar` that runs concurrently with Next.js dev server. It compiles all projects, blogs, and case-studies into a clean index file at build and dev-time.

Feel free to explore my source code on GitHub!
